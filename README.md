- Pricing in ₹ INR (Starter / Elite / Pro plans)
- Built with pure HTML, CSS & vanilla JavaScript — zero dependencies

## 🚀 Deploy on GitHub Pages (Free Hosting)

1. Create a new repository on GitHub (e.g. `iron-forge-gym`)
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Under **Source**, select `main` branch → `/ (root)` → Save
5. Your site will be live at:  
   `https://<your-username>.github.io/iron-forge-gym/`
